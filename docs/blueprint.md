# **App Name**: الحضور

## Core Features:

- Student Registration: Register new students with details like name, grade, phone, and parent contact. Generates a QR code unique to the student, and saves the data to local storage.
- Attendance Recording: Record student attendance by scanning QR codes or entering student codes, and save the attendance records locally.
- Daily Absence Report: View a report of students who are absent on a given day.
- Weekly Attendance Overview: Generate reports on student attendance trends over the past week, including those with partial attendance.
- Payment Tracking: Track student payments by recording amounts and months, offering a financial overview.
- Outstanding Payments: Identify students with outstanding payments to prompt collection efforts.

## Style Guidelines:

- Primary color: Deep blue (#0A4A8C) to evoke trust and reliability.
- Background color: Light gray (#F0F4F8) to ensure readability and a clean interface.
- Accent color: Soft teal (#45B3A9) for interactive elements, providing a gentle contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif, for a modern yet welcoming feel suitable for both headlines and body text.
- Use flat, outline-style icons for clear visual communication.
- Responsive design with a clean, card-based layout for key info blocks. RTL direction, aligning content to the right.
- Subtle transitions and loading animations for a polished feel.
